Help Center

Getting Started Guide

Welcome to AskDB! Here is how to query your data in 4 quick steps:

  1. Navigate to the Databases manager tab.
  2. Upload your custom database or use the pre-loaded E-Commerce Sample Database.
  3. Head over to the Query Workspace.
  4. Type a query in plain English and click Run (or press Ctrl+Enter).
Tip: Suggested quick prompts adapt to your active schema dynamically. Click them to auto-populate!

Example Queries

AskDB supports complex structural mapping. Try these question patterns:

Simple Lists: Show all customers
Filter Conditions: Show customers from New York
Sorting/Limits: Top 5 products by price
Joins: Show customers who ordered more than 3 times
Groupings: Count products by category
Aggregations: Average product price by category

Frequently Asked Questions

Is my database file secure?

Yes. AskDB runs queries in read-only mode, and user databases are isolated strictly within your account directory folder.

Does it support write commands?

No. For safety, write/modify operations like INSERT, UPDATE, DELETE, and DROP are blocked.

How does the AI translation work?

The copilot matches structural column keywords and query patterns against database schemas to build valid SQLite code dynamically.

Supported File Types

AskDB is optimized for SQLite databases. The following file formats are supported:

  • .db (SQLite 3 Database)
  • .sqlite
  • .sqlite3

Maximum upload size limit: 50 MB.

Export Guide

Export workspace tables in two formats:

  • CSV Export: Generates client-side structured text data for use in scripting or spreadsheet imports.
  • Excel Export: Server-side sheet generation containing header columns and clean rows.
